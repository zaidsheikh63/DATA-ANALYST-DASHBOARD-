# DATA-ANALYST-DASHBOARD-
Excited to share my latest Excel Dashboard project built using Swiggy sales data. This interactive dashboard provides key business insights through data visualization and KPI tracking
🔹 Key Metrics Included:
 • Total Sales: ₹53.0M
 • Average Rating: 4.34
 • Average Order Value: ₹268.5
 • Rating Count: 5.6M
 • Total Orders: 197.4K
🔹 Dashboard Features:
 1. Dynamic slicers for Date, Category, and Restaurant Name
 2. Monthly Sales Trend Analysis
 3. Food Type Sales Distribution (Veg vs Non-Veg)
 4. Top Performing Cities by Sales
 5. Daily & Weekly Sales Trend Tracking
 6. Clean and Interactive User Interface
🛠 Tools Used: Microsoft Excel, Pivot Tables, Pivot Charts, Slicers, KPI Cards, Data Visualization.
This project helped me strengthen my skills in data analysis, dashboard design, and transforming raw data into actionable business insights.
I would appreciate your feedback and suggestions for improvement.
